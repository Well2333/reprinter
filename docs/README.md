# PDF 构建器 — 项目概述

## 功能描述

一个运行在浏览器中的**纯前端 PDF 合并工具**，面向需要将多种来源文件（发票、合同、图片等）合并为单一 PDF 的用户。

### 核心流程

```
用户拖入文件 → 自动检测类型 → 用户确认/调整 → 设置数量 → 生成合并 PDF
```

### 功能列表

| # | 功能 | 说明 |
|---|---|---|
| 1 | 拖拽上传 | 支持拖拽或点击上传多个文件 |
| 2 | 多格式支持 | PDF、OFD、JPG、PNG、WebP、BMP |
| 3 | 类型自动识别 | 根据格式/文件名关键词判断是否为发票 |
| 4 | 用户确认类型 | 每个文件提供"发票文件"复选框供确认 |
| 5 | 指定复制数量 | 每个文件可单独设置在输出 PDF 中的份数 |
| 6 | 文件排序 | 支持拖拽排序，调整输出顺序 |
| 7 | 发票特殊处理 | 发票页面竖版居中，添加上下白边 |
| 8 | 合并输出 | 点击按钮生成并下载合并后的 PDF |

### 两种文件类型的处理规则

**非发票文件**
- 原始尺寸/方向直接合并进 PDF
- 如为图片则按图片实际比例转为 PDF 页面

**发票文件**
- 统一输出到 A4 竖版页面（210mm × 297mm）
- 内容等比缩放后在页面中**垂直居中**
- 上下留统一白边（默认各留 20mm），左右留小边距
