# 2026-06-01 — 架构修订（v2）

## 变更内容

根据用户反馈对框架设计进行修订，主要变更：

1. **框架升级**：从原生 HTML/JS 改为 **React 18 + TypeScript + Vite**
2. **新增依赖**：Zustand（状态管理）、@dnd-kit/core（拖拽排序）、Tailwind CSS（样式）
3. **新增功能**：
   - 拖拽排序（FileList 使用 dnd-kit SortableContext）
   - 合并预览（PreviewModal，展示每页含发票居中效果）
4. **发票居中算法明确化**：基于 example/in.pdf（595×409pt）→ example/out.pdf 实测推导，确认算法为等比缩放适配 A4 + 数学精确垂直水平居中
5. **OFD 策略确认**：接受矢量损失，以 2× DPI Canvas 光栅化后嵌入
6. **非发票图片**：按图片实际比例生成页面（不强制 A4）
